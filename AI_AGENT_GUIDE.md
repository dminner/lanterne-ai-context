# Lanterne AI Agent Operating Guide


---


AI agents assisting development should follow this workflow:

1. Read AI_SYSTEM_CONTEXT.md to understand architecture.
2. Read AI_PRODUCT_CONTEXT.md to understand philosophy.
3. Read AI_EXECUTION_CONTEXT.md to understand implementation.

Rules:

• ADRs override implementation assumptions.
• Product philosophy defines UX guardrails.
• System manuals define operational behavior.

Recommended development cycle:

Architecture → Implementation → Review
